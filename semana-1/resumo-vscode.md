Conteúdo a colocar:

```markdown
# Resumo VS Code

## Extensões favoritas

1. **Live Server**

   - Permite atualizar automaticamente o navegador ao salvar ficheiros HTML/CSS/JS.

2. **Prettier**

   - Formata automaticamente o código para mantê-lo organizado.

3. **GitLens**

   - Mostra histórico e autoria de cada linha de código, integrando Git no VS Code.

4. **HTML CSS Support**

   - Sugestões inteligentes para HTML e CSS.

5. **Bracket Pair Colorizer**

   - Facilita ver pares de chaves e colchetes com cores diferentes.

6. **Markdown All in One**
   - Facilita a escrita e pré-visualização de ficheiros Markdown.
```
